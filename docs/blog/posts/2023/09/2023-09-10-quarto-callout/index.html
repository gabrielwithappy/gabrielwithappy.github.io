Quarto Callout Block

Callout Block으로 내용 강조하기

Quarto
Quarto Callout으로 내용 강조하는 방법을 정리합니다.
Author

gabriel yang

Published

September 10, 2023

Quarto를 이용해 문서를 작성하는 과정에서 강조할 내용이 있는 경우 Callout Block을 사용하는 것이 좋습니다. 자산이 작성한 글에 집중이 필요한 내용을 명확하게 나타낼 수 있습니다. 공부한 내용을 정리합니다.

Callout 종류

Callout Block은 총 아래의 note, warning, important, tip, caution 5가지 종류를 갖습니다. 실제 어떻게 적용되는지 아래에서 살펴보세요.

Note

Callout note 타입을 사용합니다.

Warning

Callout warning 타입을 사용합니다.

Important

Callout important 타입을 사용합니다.

Tip

Callout tip 타입을 사용합니다.

Danger

Callout caution 타입을 사용합니다.

Markdown으로 어떻게 구현하는 지 확인해봅시다. ::: 키워드를 사용했고 Callout Block형식은 {}로 표시합니다.

::: {.callout-caution}
# 제목을 추가합니다.
Callout `caution` 타입을 사용합니다.
:::
제목을 추가합니다.

Callout caution 타입을 사용합니다.

다양한 사용자 설정

제목 추가하기

아이콘은 유지하고 제목을 변경하고 싶은 경우가 있습니다. 이 경우 header 마크다운을 사용합니다.

::: {.callout-caution}
# 제목을 추가합니다.
Callout `caution` 타입을 사용합니다.
:::
제목을 추가합니다.

Callout caution 타입을 사용합니다.

형태 변경하기

Callout Block의 외형은 default, simple, minimam 형태를 지원합니다. 각각이 어떻게 표현되는지 알아봅니다.

Default 외형

default 외형을 확인합니다.

simple 외형

simple 외형을 확인합니다.

minimal 외형

minimal 외형을 확인합니다.

코드는 아래와 같이 사용합니다. appearance가 추가되었습니다.

::: {.callout-note appearance="minimal"}
# Default 외형
default 외형을 확인합니다.
:::
