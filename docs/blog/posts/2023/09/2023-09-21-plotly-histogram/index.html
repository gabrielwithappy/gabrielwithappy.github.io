Plotly Histogram Plot만들기

tips데이터는 고객 정보와 고객이 지불한 금액과 팁을 표시한 데이터입니다.

import plotly.express as px
df = px.data.tips()
df.head(3)
total_bill tip sex smoker day time size
0 16.99 1.01 Female No Sun Dinner 2
1 10.34 1.66 Male No Sun Dinner 3
2 21.01 3.50 Male No Sun Dinner 3

서브플롯에 그래프 표시하기

’total_bill’에 대한 histogram을 표시합니다. 시간별 차이를 비교하기 위해서 facet_row에 time정보를 전달합니다.

fig = px.histogram(df, x="total_bill", facet_row="time")
fig.show()

히스토그램의 막대 갯수는 nbins로 수정합니다.

fig = px.histogram(df, x="total_bill", nbins=100)
fig.show()
