Transportation

In Korea’s big cities, it is convenient to use the subway to get around. Let’s learn how to find nearest subway.

Where is the nearest subway station?

가까운 지하철 역이 어디인가요?

You can use 어디인가요? when looking for a location. The object to find the location is a subway station, which means 지하철 역 in Korean. We use 가까운 to find the nearest subway station. Use ‘가까운’ in front of the word 지하철역 to give additional information about the ‘subway station’. Now it has the meaning of a subway station close to your location.
